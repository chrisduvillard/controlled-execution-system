# Code of Conduct

## Our Pledge

We as maintainers and contributors pledge to make participation in the
Controlled Execution System (CES) project a welcoming, respectful, and
productive experience for everyone, regardless of background or experience
level.

## Our Standards

This project adopts the [Contributor Covenant, version 2.1][covenant], a
widely used code of conduct for open source communities. The full text is
available at the link below.

- Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
- Translations: https://www.contributor-covenant.org/translations

In short: be respectful, assume good faith, focus on what is best for the
community, and show empathy toward other community members. Behavior that
degrades the collaborative environment — personal attacks, demeaning comments,
or other forms of hostile conduct — is not acceptable.

## Scope

This Code of Conduct applies within all project spaces (issues, pull
requests, discussions, commits, and any official communication channel), as
well as when an individual is representing the project in public spaces.

## Enforcement

Concerns about conduct in the project may be reported by contacting the
project maintainer, [Christophe Duvillard][maintainer], through GitHub.

All reports are handled confidentially. Maintainers are obligated to respect
the privacy and safety of the reporter. The maintainers will review and
investigate reported incidents and respond in a way deemed appropriate to the
circumstances, which may include private warnings, temporary bans, or
permanent removal from project spaces.
