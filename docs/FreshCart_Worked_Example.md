# FreshCart Worked Example

This page is a historical operating-model archive pointer, not the current shipped CES product contract.

CES is **not a sandbox** and not a hosted control plane. CES is a local control, evidence, review, and recovery layer around Codex CLI or Claude Code execution. Runtime credentials, runtime sandboxing, source-control protections, deployments, and final operator judgment remain outside CES.

The original FreshCart walkthrough describes an earlier Agent-Native Software Delivery operating-model concept with broader lifecycle, deployment, and review structures than the current local-first CES package ships. It is retained for design-history continuity only.
