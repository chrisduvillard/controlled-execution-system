# CES Demo Video — Visual Identity

## Style Prompt
Calm executive technical demo: dark navy command center, restrained champagne highlights, crisp terminal glass, evidence-led governance. Futuristic but discreet — not cyberpunk.

## Colors
- `#071018` — deep obsidian canvas
- `#0D1B2A` — navy surface
- `#12263A` — elevated panel
- `#E7C77E` — champagne proof / approval highlight
- `#64D2FF` — runtime signal blue
- `#C8D3E0` — primary text
- `#7D8FA3` — muted metadata
- `#3CEFA3` — verified state

## Typography
- System UI stack: `Inter`, `SF Pro Display`, `Segoe UI`, `Arial`, sans-serif
- Monospace stack: `JetBrains Mono`, `SFMono-Regular`, `Consolas`, monospace

## Motion Rules
- Measured 30fps motion, no frantic kinetic type.
- Cards should feel like governed artifacts entering an audit trail.
- Use champagne only for proof/approval, blue only for runtime activity.
- Prefer soft glows, thin rules, and grid motion over loud gradients.

## What NOT to Do
- No generic startup-blue hero blobs.
- No chaotic terminal rain.
- No meme or hype language.
- No red/green trading dashboard aesthetic.
- No unsupported claims about sandboxing or autonomous safety.
